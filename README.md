# 📊 محاسبه‌گر عدم قطعیت پیشرفته

یک ابزار وب حرفه‌ای و دقیق برای محاسبه عدم قطعیت اندازه‌گیری بر اساس استانداردهای GUM (Guide to the Expression of Uncertainty in Measurement).

![مجوز](https://img.shields.io/badge/license-MIT-yellow)

---

## ✨ ویژگی‌ها

- **محاسبه خودکار عدم قطعیت** بر اساس فرمول‌های استاندارد
- **پشتیبانی از پارامترهای از پیش تعریف شده**: چگالی، تنش، مدول الاستیسیته
- **پارامترهای سفارشی**: قابلیت تعریف هر فرمول دلخواه
- **تست‌های تکراری**: ورود چندین تست برای محاسبه انحراف معیار
- **منابع خطای متعدد**: رزولوشن، کالیبراسیون، دریفت و منابع اضافی
- **خروجی تفصیلی**: نمایش کامل معادلات و مراحل محاسبه
- **خروجی اکسل**: امکان ذخیره نتایج در فایل Excel
- **طراحی واکنش‌گرا**: سازگار با تمام دستگاه‌ها (موبایل، تبلت، دسکتاپ)
- **PWA Ready**: قابلیت نصب به عنوان اپلیکیشن روی گوشی
